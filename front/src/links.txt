instalando os pacotes do NODE_MODULES
npm install
(caso não tenha no seu pc o pacote node para usar os comandos dele, siga o link <https://nodejs.org/en/download/prebuilt-installer>)
================================================================

instalando o react router dom para as rotas do site e caminhos das paginas
npm install react-router-dom

instalando o firebase para funcionar o autenticador
npm install firebase

Instalando a dependencia do Slide 
npm install react-responsive-carousel

Instalar o pacote @fortawesome/react-fontawesome, que é o principal para usar ícones do FontAwesome no React.
npm install --save @fortawesome/react-fontawesome

Ícones gratuitos do FontAwesome (versão sólida)
npm install --save @fortawesome/free-solid-svg-icons

Alertas personalizados
npm install sweetalert2

================================================================

Para iniciar o projeto
npm run dev
